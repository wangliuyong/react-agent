# 灵犀 使用手册

跨平台桌面 AI 发布助手（Electron + React），支持 **小红书 / 抖音图文发布**、配图抓取与定时任务。  
大模型默认对接 **阿里云百炼（DashScope）** OpenAI 兼容接口；数据全部缓存在本机，不使用数据库。

---

## 1. 环境要求

- Node.js ≥ 20（推荐 `nvm use 20`）
- pnpm ≥ 9
- macOS / Windows / Linux

## 2. 安装与启动

```bash
cd react-agent
pnpm install
pnpm install:browser   # 经 npmmirror 加速下载 Playwright Chromium
pnpm dev
```

`pnpm install:browser` 默认走国内镜像（npmmirror）。若镜像异常，可用官方源：`pnpm install:browser:official`。

## 3. 配置阿里云百炼

1. 打开 [阿里云百炼控制台](https://bailian.console.aliyun.com/) 创建 API Key。  
2. 启动应用后进入左侧 **设置**。  
3. 填写：
   - **DASHSCOPE API Key**
   - **Base URL**（默认 `https://dashscope.aliyuncs.com/compatible-mode/v1`）
   - **模型**（默认 `qwen-plus`，可改为 `qwen-max` / `qwen-turbo` 等）
4. 点击保存。

> Key 仅保存在 Electron `userData/react-agent-data/settings.json`，不会提交到 git。

也可参考仓库根目录 [`.env.example`](../.env.example)（当前版本以应用内设置为准）。

## 4. 发一条小红书（推荐路径）

1. 点击 **+ 新任务**。  
2. 输入例如：

   > 帮我发一条小红书，内容关于今日 A 股收盘速读，标题不超过 20 字。请从相关新闻网页抓取配图后发布。

3. Agent 会优先调用 `fetch_web_images` 从来源页下载配图（也可在 `xhs_publish_note` 里传 `imageSourceUrl`）。  
4. **本地上传为可选**：需要指定某张图时，再点输入框旁回形针选择图片。  
5. 右侧会打开 **智能体浏览器**，进入创作台；未登录时按提示扫码后点 **继续**。  
6. 非「完全访问」模式下，正式点「发布」前会再次确认。  

### 配图优先级

1. `imagePaths` / `fetch_web_images` 下载的本地文件  
2. `imageSourceUrl` / `imageUrls`（工具内自动下载）  
3. 用户本轮上传的附件（可选）

### 权限说明

| 模式 | 行为 |
|------|------|
| 需确认（默认） | 登录、正式发布前暂停等人 |
| 完全访问 | 跳过部分确认（仍建议人工看一眼发布结果） |

## 5. 发布工作台

1. 侧栏进入 **发布**。  
2. 可点 **导入示例** 得到「小红书任务」双生子任务。  
3. 编辑子任务的主题与内容说明。  
4. 点 **运行**：会新建主聊天会话，由 Agent **串行**执行子任务。  
5. 执行进度请在主聊天窗口与任务清单中查看。

## 6. 本地数据目录

应用数据位于系统 userData 下的 `react-agent-data/`：

| 路径 | 内容 |
|------|------|
| `settings.json` | API Key、模型、权限 |
| `sessions/` | 聊天会话 |
| `publish-plans/` | 发布计划 |
| `browser-profile/` | 小红书登录 Cookie（Playwright Profile） |
| `artifacts/` | Agent 写入的本地文件 |

清除登录：设置页 → **清除小红书登录态**。

## 7. 架构要点（对齐 Claude Code）

- **单一 ReAct 循环**：模型产出 `tool_calls` → 主进程执行工具 → 结果回灌，直到结束。  
- **工具注册表**：文件 / 浏览器原子操作 / `xhs_publish_note` 技能脚本。  
- **智能体浏览器**：Playwright 持久化上下文 + 截帧预览。  
- **前端按业务域拆分**：`src/features/chat|publish|settings|browser`。

## 8. 常见问题

**Q: 提示未配置 API Key**  
A: 先到设置页保存 Key。

**Q: 上传控件找不到 / 填不了标题**  
A: 小红书 DOM 可能改版。Agent 会回退到 `browser_snapshot` + 原子工具；也可在右侧浏览器手动补全后点继续。

**Q: Chromium 弹窗和右侧预览同时出现？**  
A: 当前为有头 Playwright 便于登录与调试，右侧同步截帧。后续可改为仅嵌入预览。

**Q: 提示「正在现有的浏览器会话中打开」或 browser has been closed？**  
A: 多为上次异常退出留下的 Profile 锁。应用会在启动浏览器前自动清理 `SingletonLock` 并结束残留的 Chrome for Testing；请重启 `pnpm dev` 后重试。若仍失败，可到设置页点「清除小红书登录态」。

**Q: 文档生成物放哪？**  
A: 项目约定一律放在本仓库 [`doc/`](./) 目录。
