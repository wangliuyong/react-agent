# AI 视频制作工作流

## 适用场景
- 用户要求生成一段叙事性视频（如猫捉老鼠、童话故事等）
- 需要从创意到成片的端到端自动生产
- 素材可独立产出，最后合成

## 标准任务清单

1. **创意剧本创作**
   - 根据主题创作完整剧本（通常 4-6 场）
   - 输出剧本文本

2. **分镜设计**
   - 将剧本拆解为多个镜头（如 6 镜）
   - 为每个镜头设计画面描述、镜头类型、时长
   - 输出分镜清单（storyboard.json）

3. **关键帧生成**
   - 为每个镜头生成一张关键帧图片（如 shot-01.png ~ shot-06.png）
   - 使用图像生成工具，确保风格统一

4. **场景动效视频生成**
   - 为每个镜头生成一段带动效的视频片段（如 shot-01.mp4 ~ shot-06.mp4）
   - 可基于关键帧或直接生成

5. **旁白录制**
   - 为每个镜头录制对应的旁白音频（如 shot-01.wav ~ shot-06.wav）
   - 确保语音与画面节奏匹配

6. **合成成片**
   - 合并所有视频片段、旁白音频，添加转场和背景音乐
   - 输出最终视频文件

## 工具与参数
- `compose_video`：合成工具，需注意 ffmpeg 参数顺序（`-filter:v` 必须放在输入文件之前，否则会报错）
- 图像生成：推荐使用 DALL-E 或 Stable Diffusion
- 视频生成：推荐使用 RunwayML 或 Pika
- 音频：推荐使用 ElevenLabs 或 Azure TTS

## 注意事项 / 常见问题
- **ffmpeg 参数顺序**：合成时 `-filter:v` 必须放在输入文件之前，否则会报错 "you are trying to apply an input option to an output file"。
- **素材命名规范**：保持场景序号一致，便于后续合成。
- **分镜清单**：建议使用 JSON 文件记录每个镜头的参数和路径，便于合成脚本读取。
- **失败处理**：如果合成失败，检查素材文件是否完整，并修复 ffmpeg 命令。
